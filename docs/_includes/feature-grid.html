Read

Search OpenAlex & arXiv, read PDFs and LaTeX source, build .bib files, and fact-check claims with /research.

Run

Each Python experiment runs in an isolated uv project, captured as a reproducible capsule — code, env, metrics, repro.sh.

Write

Draft from ACL, NeurIPS, or blank template. Edit in place with a diff box. Sync two-way with Overleaf.

Prove

A claim ledger, /audit-paper, /provenance, and /handoff packets keep every number traceable to its evidence.

Integrate

Annotate papers in Zotero with PDF highlights. Fetch reviewer feedback from OpenReview and draft responses.

Local-first

Models run on your machine. Your literature, data, and drafts stay with you. Cloud only if you explicitly opt in.
